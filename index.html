Trabalhos

Trabalho 1

Objetivo: Desenvolver uma tela de apresentação hard skills e soft skills.

Comandos:O código cria uma tela de perfil usando React Native, estruturada dentro de um SafeAreaView para respeitar os limites do dispositivo e um ScrollView para permitir rolagem do conteúdo. No topo, é exibido o nome do usuário, seguido de uma imagem de perfil estilizada dentro de um container com bordas arredondadas. Em seguida, há uma breve descrição e uma seção de habilidades organizada com flexDirection: "row", que posiciona dois cartões lado a lado: um para soft skills e outro para hard skills. Cada cartão contém um título e uma lista de habilidades. Por fim, há um rodapé com um texto de identificação da atividade. A estilização é feita com StyleSheet, definindo cores, tamanhos, espaçamentos e alinhamentos para criar um layout visual organizado e moderno.

Trabalho 2

Objetivo: qFlexDiretion RPG .

Comandos: O código cria um card de personagem de RPG utilizando React Native, organizado dentro de um SafeAreaView para garantir que o conteúdo respeite os limites da tela e um ScrollView para permitir rolagem. No topo, é exibido o nome do personagem, seguido de uma imagem representando seu visual dentro de um container estilizado com bordas arredondadas. Abaixo, há uma breve descrição da classe do personagem. Em seguida, os atributos são organizados utilizando flexDirection: "row", permitindo que duas seções fiquem lado a lado, como status (força, agilidade, inteligência) e habilidades especiais. Cada seção é exibida em formato de card, contendo título e lista de informações. Por fim, há um rodapé com detalhes adicionais, como nível ou campanha. A estilização é feita com StyleSheet, definindo cores, espaçamentos e alinhamentos para criar um visual temático e organizado, semelhante a um card de RPG..

Trabalho 3

Objetivo: Descreva aqui o objetivo do trabalho.

Comandos: O código utiliza o componente TouchableOpacity do React Native para criar um botão interativo que responde ao toque do usuário. Esse componente envolve um elemento visual, geralmente um Text ou View, e aplica um efeito de opacidade quando pressionado, dando um feedback visual de interação. Dentro dele, é definida uma função no evento onPress, que é executada quando o botão é clicado, podendo realizar ações como exibir mensagens, navegar entre telas ou atualizar dados. A estilização do botão é feita com StyleSheet, definindo propriedades como cor de fundo, espaçamento, bordas e alinhamento do texto, garantindo um visual organizado e agradável. Dessa forma, o TouchableOpacity permite criar botões simples e funcionais, melhorando a interatividade da aplicação..

Trabalho 4

Objetivo: Descreva aqui o objetivo do trabalho.

Comandos: O código implementa um aplicativo em React Native utilizando Expo que apresenta um sistema de paginação com renderização condicional entre telas. Inicialmente, é exibido um menu com botões representando os filmes em cartaz. Cada botão, ao ser pressionado, aciona uma função através do evento onPress, que altera o estado da aplicação para identificar qual filme foi selecionado. Com base nesse estado, ocorre a renderização de uma nova tela que exibe a imagem do filme e um texto descritivo com suas informações. Essa troca de conteúdo é feita sem navegação externa, apenas utilizando renderização condicional dentro do próprio componente. Além disso, há um botão “voltar”, também implementado com TouchableOpacity, que redefine o estado e retorna à tela inicial com a lista de filmes. A estrutura utiliza componentes como View, Text, Image e ScrollView, enquanto o StyleSheet é responsável pela organização visual, garantindo um layout claro, funcional e agradável ao usuário.
